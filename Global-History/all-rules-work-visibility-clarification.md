# Work Visibility - Critical Clarification
**Date**: 2025-01-29  
**Status**: MANDATORY Understanding

---

## Critical Understanding

**"Show your work in the session window"** means:
- ✅ Display ALL commands as they execute
- ✅ Display ALL command outputs immediately
- ✅ Display compilation results as they happen
- ✅ Display test results as tests complete
- ✅ Display progress updates continuously
- ✅ Display timestamps for every operation

**NOT**:
- ❌ Show modified files at the end of milestones
- ❌ Show summaries only
- ❌ Hide intermediate steps
- ❌ Wait to show results

---

## Enforcement

This clarification is MANDATORY for:
- ALL work sessions
- ALL milestones
- ALL task completion
- ALL testing
- ALL builds

---

**Purpose**: User needs real-time visibility to ensure work is actually happening and to catch issues early.
