- Decide what you want readers to feel
- Consult emotional techniques guide
- Select elements that support that emotion
- Build toward emotional peaks

### 2. **Layer Your Elements**
- Combine magic + creatures + human elements
- Create complex, rich scenes
- Avoid single-note moments
- Depth through combination

### 3. **Subvert Expectations**
- Know the tropes (from compendia)
- Then twist them
- Surprise while remaining coherent
- Innovation informed by tradition

### 4. **Create Contrast**
- Small light in great darkness (powerful)
- Humor in tense moment (relief)
- Beauty in horror (uncanny)
- Contrast amplifies both elements

### 5. **Earn Your Moments**
- Big emotions need setup
- Develop relationships before testing them
- Establish normal before breaking it
- Payoff proportional to setup

### 6. **Remember the Human Core**
- Even fantasy needs human truth
- Emotions must be genuine
- Relationships must feel real
- Readers connect to humanity, even in monsters
