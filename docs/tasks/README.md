## TASK FORMAT

Each task includes:
- **Task ID**: Unique identifier
- **Title**: Clear task name
- **Description**: Detailed requirements
- **Acceptance Criteria**: How to verify completion
- **Estimated Time**: Duration estimate
- **Dependencies**: Prerequisites
- **Watchdog**: Command protection protocol
- **Testing**: Test requirements (mandatory)

---

## INTEGRATION WITH COMMANDS

- `/all-rules`: Enforces all coding standards
- `/autonomous`: Enables autonomous execution
- `/complete-everything`: Complete all tasks
- `/test-comprehensive`: Full test suite
- `/test-end-user`: End-user testing with Playwright

---

## MILESTONE INTEGRATION

Every task completion:
1. Runs comprehensive tests
2. Consolidates learnings to project/global memory
3. Back-tests everything built
4. Ensures nothing breaks
